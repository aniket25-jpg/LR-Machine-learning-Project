This project focuses on building a machine learning model to predict delivery times for a logistics application using key operational features such as delivery distance, order volume, and time of day. Accurate delivery time prediction is critical for improving customer experience, planning logistics operations, and optimizing resource allocation.

A Linear Regression model is implemented as a baseline predictive solution, supported by proper data preprocessing, feature engineering, and performance evaluation.
